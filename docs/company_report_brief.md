**방법론**

- 계획 공기: 12개월

**핵심 비교 지표**

- 비용 증가 (직접비용 + 금융비용)
- 금융 리스크 (PF 금리 변동)

## ✅ 주요 완료 사항

### 1. 시스템 아키텍처 구축

**BIM 품질 지표 체계 확립**
- 경고밀도(WD): Revit 경고 건수 / 연면적
- 충돌밀도(CD): 간섭 건수 / 연면적
- 속성 채움률(AF): 필수 속성 입력 비율
- Phase 연결률(PL): 3D 모델-일정 연계 비율

**이슈 카드 데이터베이스 구축**
- 총 27개 이슈 카드 정의
- 단계별 분류: 설계(5) / 시공(12) / 발주(4) / 자재(4) / 감리(2)
- 각 이슈별 기본 영향(지연, 비용), BIM 탐지 가능성, 발생 확률 설정

**멀티 에이전트 시스템 구현**
- 건축주: 예산/일정 최우선, 최종 의사결정권자
- 설계사: 기술적 해결책 제시, BIM 모델 관리
- 시공사: 현장 실행 가능성 검토, 공정 관리
- 감리사: 중립적 품질/안전 검증
- 금융사: 리스크 평가 및 금리 조정

### 2. 핵심 알고리즘 개선

**이슈별 맞춤형 BIM 효과성 계산**
- 기존: 모든 이슈에 동일한 가중치 적용 (WD 25%, CD 30%, AF 25%, PL 20%)
- 개선: 이슈 특성에 맞는 차등 가중치 적용
  - 예) 설비-구조 간섭 → 충돌밀도(CD) 70% 집중
  - 예) RFI 폭증 → 속성 채움률(AF) 60% 집중

**비선형 탐지 확률 모델 (Sigmoid 함수)**
- 기존: 선형 관계 (품질 0.5 → 탐지율 50%)
- 개선: S자 곡선 적용 (품질 임계값 이후 급격한 효과 증가)
- 현실 반영: "낮은 품질 BIM은 거의 안 쓰다가, 일정 수준 넘으면 적극 활용"

**발견 시점별 차등 영향 계산**
- 설계 단계 발견: 95% 지연 감소, 98% 비용 감소 (도면 수정만)
- 시공 초기: 70% 감소
- 시공 중기: 50% 감소 (일부 재시공)
- 시공 후기: 25% 감소 (전면 철거)

**금융 비용 계산 추가**
- PF 대출 금리 인상 (지연 기간별: 0bp ~ 100bp)
- 간접비 누적 (현장 관리비, 장비 대여료 등 일 0.1%)
- 직접 비용만 계산하던 기존 방식 대비 현실성 대폭 향상

### 3. 버그 수정 및 안정화

**이슈 발생 동일성 보장**
- 문제: BIM ON/OFF 시나리오에서 서로 다른 이슈 발생
- 해결: Random Seed 로직 수정으로 동일 이슈 보장
- 검증: 200일 테스트에서 12개 이슈 동일 날짜 발생 확인 ✅

**그래프 한글 폰트 처리**
- Matplotlib 맑은 고딕 폰트 적용
- 마이너스 기호 깨짐 방지 설정

## 🔜 향후 계획

### 1. 검증 및 보정 (1-2주)
- 업계 벤치마크 데이터와 비교 (일반 프로젝트 22% 예산 초과, BIM 프로젝트 재시공 49% 감소)
- 전문가 인터뷰를 통한 시나리오 현실성 검증
- 민감도 분석 (파라미터 변화에 따른 결과 변동 확인)

### 2. 최종 시뮬레이션 실행
- 다양한 BIM 품질 수준별 시뮬레이션 (Excellent/Good/Average/Poor)
- 프로젝트 템플릿별 비교 (오피스텔, 아파트, 상업시설 등)

### 3. 결과 보고서 작성
- 경영진 보고용 요약본
- 기술팀용 상세 보고서
- 시각화 자료 (그래프, 인포그래픽)

## 💡 기대 효과

**1. 의사결정 지원**
BIM 도입 투자 대비 효과를 정량적으로 제시하여 경영 판단 근거 마련

**2. 발주처 제안 자료**
발주자에게 BIM 적용 필요성을 구체적 수치로 설득 가능

**3. 프로젝트 리스크 관리**
BIM 품질 지표별 목표 설정 → 실시간 모니터링 → 리스크 사전 대응

**4. 사내 BIM 품질 기준 수립**
4가지 품질 지표를 사내 BIM 표준으로 활용 가능
